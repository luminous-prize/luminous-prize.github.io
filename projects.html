Credit Default Risk Modeling

Johns Hopkins · Quantitative Modeling

Developed a credit default prediction model using regularized logistic regression, emphasizing temporal stability, asymmetric error costs, and decision-aware calibration rather than raw accuracy.

  • Engineered behaviorally motivated features from utilization, payment history, and exposure dynamics
  • Achieved AUC 0.82 and KS 0.41, improving rank-ordering power by 17% over a naïve baseline
  • Evaluated performance under temporal splits and cost-sensitive objectives
  • Reduced false negatives by 24% at fixed precision, reflecting real credit risk asymmetry

Python, Logistic Regression, Risk Modeling, Statistical Evaluation

IBM Storage Insights MCP Server

Open Source · Agentic AI · Observability Systems

Open-sourced a Model Context Protocol (MCP) server that exposes IBM Storage Insights observability capabilities to agent-based AI workflows. The server enables external AI agents to query, diagnose, and reason over storage telemetry through a structured, extensible interface.

  • Designed a Python-based MCP server exposing 13 observability tools backed by Storage Insights APIs
  • Enabled seamless integration with agent frameworks and Claude Desktop via stdio and streamable HTTP
  • Engineered multi-tenant credential handling, structured logging, and extensible API routing
  • Reduced manual diagnosis latency by enabling autonomous, tool-using agents for storage health analysis

Python, MCP, LLM Tooling, Observability APIs

Ransomware Threat Detection for FlashSystem

IBM · ML Systems · Time-Series Security

Built a production-grade ransomware detection system using ensemble classifiers trained on over 500k FlashCore Module filesystem traces. The system was designed to generalize across ransomware families while maintaining an extremely low false-positive rate on live workloads.

  • Trained SnapML-based ensemble models with automated re-balancing and cross-family validation
  • Achieved <1% false-positive rate on production workloads
  • Integrated inference into the Storage Virtualize stack to trigger early alerts and immutable snapshots

Python, SnapML, Scikit-learn, Time-Series Analysis

LLM Sensitivity to Politeness and Emphasis

Johns Hopkins · LLM Evaluation

Studied how linguistic framing affects reasoning reliability and compute usage in modern LLMs. Evaluated multiple prompt variants on GSM8K and quantified the accuracy–compute tradeoff using paired statistical tests.

  • Evaluated Llama 3, GPT-5, and Claude Sonnet on controlled linguistic variants
  • Observed <1.5% accuracy variance but 12–18% higher token usage
  • Validated results with McNemar tests, t-tests, and effect-size analysis

Python, LLM Evaluation, Statistical Testing
